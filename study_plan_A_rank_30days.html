📘 30日プランの設計方針

  • 1日3語ペース（最終日Day 30は新規2語＋全体総復習）。1日約20分の学習を想定。
  • 復習日は Day 10・Day 20・Day 30 の3回。直前9日分（または全体）を一気に確認するためのチェックポイント。
  • 順序の意図：Phase 1〜2 (Day 1-9) で企画とゲーム性の骨格 → Phase 3 (Day 11-19) で UI/UX・体験設計・意思決定 → Phase 4 (Day 21-29) で運営/KPI・分析・上級概念。
  • 使う PDF は8つ: batch_001 / 002 / 003 / 004 / 005 / 013 / 018 / 028 / 038 / 048 / 058 / 078（全A語をカバー）。各日の PDF 列のリンクから直接開けます。
  • 進捗チェック：右端のチェックボックスで「その日完了」を記録。study_dashboard.html 側の「実戦で使える」状態とも連携可能（「『実戦で使える』状態と連動」ボタン）。
